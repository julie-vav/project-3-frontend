# Introduction 

This app was inspired by JIRA, a popular ticketing system most companies use. This a simple app meant for users to be able to create a ticket and read it. 

# Technologies Used

- React
- Express
- Node
- MongoDB
- Mongoose
- HTML 5 
- CSS
